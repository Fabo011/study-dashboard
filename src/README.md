
## Installation
Make sure that you have python and pip installed on your machine.

```bash
python3 -m venv venv
```

### Avtivate env

Windows
```bash
venv\Scripts\activate
```

Linux, Mac
```bash
source venv/bin/activate
```

### Install requirements

Install requirements
```bash
pip3 install -r requirements.txt
```

---

<br>

## Misc
If you want to deactivate dev env
```bash
deactivate
```
